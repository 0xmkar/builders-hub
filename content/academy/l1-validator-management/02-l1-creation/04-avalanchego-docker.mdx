### Follow Node Logs

```bash
docker logs -f avago
```


### Stop Node

```bash
docker stop avago
```

### Remove Node

```bash
docker rm avago
```

### Run Signature Aggregator
```bash
docker run -d --network=host -v $(pwd)/.devcontainer/sig-agg-config.json:/sig-agg-config.json avaplatform/signature-aggregator --config-file=/sig-agg-config.json
```
